Keep learning with the /teach skill

Lessons are generated and taught interactively with Matt Pocock's teach skill for Claude Code. Install it once:

npx skills add https://github.com/mattpocock/skills --skill teach

Then open this workspace in Claude Code and run /teach — it reads MISSION.md and NOTES.md to pick up where you left off. Your per-lesson progress above is stored locally in this browser.
